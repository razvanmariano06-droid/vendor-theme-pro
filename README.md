# Vendor Theme Pro

A professional-grade Shopify 2.0 vendor/supplier reselling theme.

## Theme Info
- **Theme Name:** Vendor Theme Pro
- **Version:** 1.0.0
- **Platform:** Shopify 2.0

## Features
- Fully customizable from Shopify Theme Customizer
- 15 color pickers
- Dynamic typography settings
- Animated product cards with spinning borders
- CSS-only carousels
- Full customer account pages
- Responsive design
- Translation-ready (i18n)
- Black and pink gradient default color scheme
